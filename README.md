# Super Vasco

Jogo Super Vasco numa aplicação web com **Phaser 3** e **Vite**.

## Como executar

```bash
npm install
npm run dev
```

Abre o URL indicado no terminal (ex.: http://localhost:5173).

## Build

```bash
npm run build
```

A saída fica na pasta `dist/`. Para testar em local: `npm run preview`.

## Controlos

- **Esquerda / Direita**: setas ou A / D  
- **Salto**: seta cima, W ou Espaço  

## Objetivo

- Recolher **todas as moedas** do nível e alcançar a **bandeira verde** à direita para vencer.  
- Evitar os Goombas: pisa em cima para os eliminar; colidir de lado = Game Over.  
- Carrega em **R** para reiniciar após vitória ou game over.
